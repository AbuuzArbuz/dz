Написать 2 программы: server и client.   
1) client. Должен запускаться с 3мя аргументами: a, b и id. 
Id - идентификатор клиента для сервера, чтобы сервер знал куда отвечать. 
Отправляет запрос, содержащий числа a,b,id через очередь сообщений на сервер и в этой же очереди ждет ответа. После получения ответа завершается.
2) server, главный поток. Работает все время. Читает очередь сообщений и запускает отдельный поток (процесс или тред на выш вкус) для обработки запроса, после чего ждет новых запросов. Если в данный момент этот id уже в обработке, или в обработке больше 2ух запросов - сразу сообщить клиенту ошибку, вместо создания вспомогательного потока.
3) server, вспомогательный поток. Делает вид, что думает - спит a + b секунд. После чего отправляет клиенту ответ - сумму a и b.
Обо всех действиях все программы должны печатать на экран.
